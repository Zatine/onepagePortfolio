ScreenDance Festival

An upgrade of ScreenDance Festival's website. Made together with Emma Berglund using Wordpress.

Tasks included designing and building a theme from scratch with little prior knowledge of PHP or Wordpress, making it easy to update and keeping the website responsive for different devices. To make the work slightly easier, we used the grid-framework Skeleton.

Snake

While learning JavaScript in school we made a "Lights out"-game by creating a table and assigning different ID's to all the cells. It was a fun challenge, and I thought it might be possible to create a more advanced game with the same technique. I realized the classic game of Snake would be perfect, and probably suit my current skill level at the time.

It's not perfect, but I'm definitely satisfied and think it works well. I'm also happy with the graphics I made, even if the styling isn't completely finished yet.

Play Snake here!
